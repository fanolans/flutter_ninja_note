- Edit Note

## Tech Stack

**Front-End:** Flutter

**Back-End:** Firebase

## Run Locally

Clone the project

```bash
  git clone https://github.com/fanolans/ninja-note.git
```

Go to the project directory

```bash
  cd project-folder
```

Install dependencies

```bash
  flutter pub get
```

Start the app

```bash
  flutter run
```
